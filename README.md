basically a simple index card directory/utility/app for exchanging food cans for cash or
cash for food cans. A bartering database for a last resort way to acquire some survival resources:
namely cash or food requirements.
-App serves to specific local areas by postal code. Creates 1 page card directory for each postal code
-all cards expire in 24 hours from the time they are created.
-Create/delete cards  (download card code if you decide to delete it yourself)
-make offers negotiable 
-describe your item or state your offer
-schedule meetups
